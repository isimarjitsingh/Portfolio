# Simarjit Singh - Portfolio

A modern, responsive portfolio website built with React.js and Tailwind CSS featuring attractive black gradient combinations.

## Features

- **Hero Section**: Personal introduction with contact links
- **Skills Section**: Technical skills with progress bars
- **Projects Section**: Project showcase with detailed descriptions
- **Hackathons Section**: Achievements and hackathon certificates
- **Contact Section**: Contact information and links
- **Responsive Design**: Works perfectly on all devices
- **Modern UI**: Black gradient theme with smooth animations

## Technologies Used

- React.js
- Tailwind CSS
- Vite
- PostCSS
- Autoprefixer

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
